# Girmitiya Foundation Premium NGO Website

A complete React + Vite NGO website for **Girmitiya Foundation**, built with a fresh premium cultural design direction inspired by Girmitiya heritage, community care, and foundation-style storytelling.

## Stack

- React.js with Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Lucide React icons
- Swiper.js

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Pages

Home, About Us, Our Team, Our Mission, Membership, Blog, Activity, Woman Empowerment, Skill Development, Child Welfare, Publication, Root Search, and Contact.

## Notes

The design uses CSS gradients and optimized remote image URLs for polished placeholders. Replace the image URLs and contact details with official foundation assets before production launch.
